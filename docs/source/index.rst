``kaplanmeier`` is a Python package to compute the kaplan meier curves together with the log-rank test, and make plots.



-----------------------------------

.. note::
	**Your ❤️ is important to keep maintaining this package.** You can `support <https://erdogant.github.io/kaplanmeier/pages/html/Documentation.html>`_ in various ways, have a look at the `sponser page <https://erdogant.github.io/kaplanmeier/pages/html/Documentation.html>`_.
	Report bugs, issues and feature extensions at `github <https://github.com/erdogant/kaplanmeier/>`_ page.

	.. code-block:: console

	   pip install kaplanmeier
