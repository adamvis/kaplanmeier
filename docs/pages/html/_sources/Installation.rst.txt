Installation
####################

.. code-block:: console

    # Install from Pypi:
    pip install kaplanmeier

    # Force update to latest version
    pip install -U kaplanmeier



Remove installation
######################

Note that the removal of the environment will also remove the ``kaplanmeier`` installation.

.. code-block:: console

    # Install from Pypi:
    pip uninstall kaplanmeier
